Arduino IDE: Touch Input

ABOUT THE TUTORIAL
This tutorial discuss how to get capacitive touch input in Arduino IDE using Adafruit MPR121 library.
TUTORIAL INFO
Programming Platform: Arduino IDE Type: Tutorial Difficulty: Easy Estimated Time: 1.5 hours

Introduction

evive has 12 touch outputs, which are interfaced via the MPR121 chip using SPI interface. These touch inputs can be attracted to any fruit or metal surface and can be used as inputs. However, these inputs are complicated to interface with Arduino IDE, but using Adafruit MPR121 library, you can easily interface the sensors.

Example:

In this example we will get input from channel 1. We will connect a banana or any other fruit you have to channel 1 and try to detect if someone has touched the banana and display it on the TFT Screen.
