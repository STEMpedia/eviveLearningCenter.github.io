Getting Started with Scratch

Scratch is a free programming language where you can create your own innovative and interactive projects, stories, games, and animations. mBlock is based on Scratch 2.0 and let you use evive for your projects through its Arduino plugins.

Installation Instructions

Installation of mblock is simple and pretty straight forward. Here we are showing the installation steps for windows user only.

  1. Go to mBlock.cc and download the program, by clicking on “Windows Download” buttons.
  2. When the download is finished, unzip the file. In the folder, you will find mBlock application setup file (.exe extension). Run the setup.
  3. Choose the appropriate language and press “OK”. In our case, we are choosing English.
  4. A new window will popup. Press “Next”.
  5. Choose the installation directory (We suggest you to not change the directory). Then press “Next”.
  6. Select start menu folder, where the setup will place the program’s shortcuts. Then press “Next”.
  7. Check the “Create a desktop icon”, if you want to create the desktop icon. Then press “Next”.
  8. Press “Install”.
  9. After the installation is finished press “Finish”.

Creating first script in Scratch

evive has a programmable LED connected to its digital pin 13. Thus, LED glows whenever digital pin 13 is HIGH, and turns off whenever it is LOW. In Scratch, we will make digital pin 13 HIGH for 1 second, then LOW for another 1 second and repeat. Open Scratch application on your desktop and start following the steps:

Step 1:
Whenever you want to run a script, you have to start with a Hat block which are present in Event category. There is a block named ‘when space key is pressed’, which starts the execution of the program when space is pressed. We will use this block to start our program. Drag the block and drop the block in script using mouse.



Step 2:
Now, we want to run the code forever. For this types of functions there is a C block named ‘Forever’, which runs the code inside it continuously in loop. Drag the block and snap it below the hat block.

Step 3:
To access the evive digital pin 13, click on Arduino tab, you have to evive via serial port. First connect evive to you PC using USB B type connect. Select the Board to “Arduino Mega 2560”
Click on Connect and select the serial port to which evive is connected.
Upload or upgrade the firmware to evive.
Finally in Extension tab select Arduino. Now you are ready to use evive digital and analog pins.
Once you select a COM Port, you will get a message “Serial port connected” at the top. If you don’t get this message, select another COM port.

Step 4:
You want to turn ON the LED. For this digital pin 13 should be HIGH, which can be done using ‘set digital pin’ block. Drag and snap the block inside forever block. Next to the pin inside the block you can see white space where you have to input the pin number (13 in this case). To turn the pin HIGH, select the option at the end of the block.

Step 5:
Now you want to wait for 1 seconds. In Control panel there is wait block. Drag and drop it inside the forever loop below the digital pin block.

Step 6:
Now you want to turn the LED OFF. For this the digital pin should be LOW. Right click on the set digital pin block inside forever loop and duplicate.
You will see two duplicated blocks, which you have to snap below wait block. Select the LOW option in set digital pin block.
Your program is ready now and you can press ‘space’ to start it. The whole script will start glowing indicating that the program is running. To stop it press ‘Esc’ on keyboard.
