TFT Colour

In the last tutorial you have seen how to write text on screen with white colour. Well what if you want to write with some other colour?

There are few colours defined in Adafruit ST7735 library:

#define ST7735_BLACK   0x0000
#define ST7735_BLUE    0x001F
#define ST7735_RED     0xF800
#define ST7735_GREEN   0x07E0
#define ST7735_CYAN    0x07FF
#define ST7735_MAGENTA  0xF81F
#define ST7735_YELLOW  0xFFE0
#define ST7735_WHITE  0xFFFF

You can set the text colour using this function:

tft.setTextColor(unit16_t color);

where color can be any of the above define colours: ST7735_BLACK, ST7735_BLUE, ST7735_RED, ST7735_GREEN, ST7735_CYAN, ST7735_MAGENTA, ST7735_YELLOW, ST7735_WHITE. These colours are not only available for this function only, there are other function also which can have these colours in their argument.
