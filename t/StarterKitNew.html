Obstacle Avoidance Robot
In this project, we will make a robot that will be able to avoid obstacles. The robot will use an ultrasonic sensor and a servo motor in addition to the basic robot. The robot will check how far the nearest obstacle is (in every direct on) and then decide upon the actions to be taken. The servo controls the direct on in which the sensor faces and if the robot is hindered by an obstacle, the servo will probe in different directions. Once we are convinced that a certain direct on is clear of any obstacles, we can turn the robot in that particular direct on and then move it in a straight line along that direct on till the next obstacle is found. If there is no way to go ahead the robot executes a full 180 degrees turn.
PROJECT INFO
Programming Platform: Scratch Type: Robot Difficulty: Intermediate Estimated Time: 5 hour
