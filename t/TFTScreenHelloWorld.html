TFT Screen: Hello World

ABOUT THE TUTORIAL
This tutorial describes how to use TFT Screen of evive and write your first program to display text on TFT screen.
TUTORIAL INFO
Programming Platform: Arduino IDE Type: Tutorial Difficulty: Easy Estimated Time: 2 hours

Introduction

Visual feedback is very useful for electronics projects and prototyping. They help you debug your code, display data and soometime make your project look cool.

evive has a 1.8" TFT Display with 128 x 160 colour pixel. This TFT uses ST7735R TFT driver to display 18-bit colour (262,144 shades). Let us see how we can use the TFT Screen in our projects.

Adafruit ST7735 TFT Library

We will use Arduino IDE to control the TFT display.

Two libraries are required for TFT:

  1. Adafruit_ST7735_Library (http://adafru.it/dXk): this contains the low-level code specific to this device.
  2. Adafruit_GFX_Library (http://adafru.it/cBB): handles graphics operations common to many displays.

Download both ZIP files, uncompress and rename the folders to Adafruit_ST7735 (contains Adafruit_ST7735.cpp and .h) and Adafruit_GFX (contains Adafruit_GFX.cpp and .h) respectively. Then place them inside your Arduino libraries folder and restart the Arduino IDE.

After restarting the Arduino software, you should see a new example folder called Adafruit_ST7735 and inside, an example called graphicstest. Change the pin details as shown below.

#define TFT_CS     48
#define TFT_RST    47
#define TFT_DC     49

Now upload the sketch to your Arduino. You should see a collection of graphical tests draw out on the TFT.

Write Text on Screen

Now, as you have tested the Display, you are ready for writing the program in Arduino IDE to display “Hello World” and a message.

In the beginning of the code you have to include libraries using these commands:

#include <Adafruit_GFX.h
#include <Adafruit_ST7735.h> 
#include <SPI.h>

SPI Library is used to communicate with TFT Screen.

Then you have to define CS, RST and DC pin numbers which are connected internally to digital pin 48, 47 and 49 respectively. These can be defined as:

#define TFT_CS     48
#define TFT_RST    47
#define TFT_DC     49

Now you have to define an object for TFT using its constructor function.

Adafruit_ST7735 tft = Adafruit_ST7735(TFT_CS,  TFT_DC, TFT_RST);

In setup loop, you have to initialise the ST7735S Chip using this function:

tft.initR(INITR_BLACKTAB);

Also it is recommended that you use tft.fillscreen(ST7735_BLACK); function to clear the screen with black background in setup function.

These are the functions that you can use to customise and write text on TFT:

Function

Description

setCursor(int16_t x, int16_t y)

Set cursor to a location

setTextColor(uint16_t c)

Set text colour

setTextColor(uint16_t c, uint16_t bg)

Set text colour

setTextSize(uint8_t s)

Set text size

setTextWrap(boolean w)

Set text wrap

print(string)

Display input string on TFT

println(string)

Display input string on TFT and move to next line


Using setCursor() function you can set the cursor to a location for next print command.

Using setTextSize() function you can set the size of the text.

Using print() and println function you can write text on the string.
