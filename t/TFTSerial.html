evive TFT Serial

ABOUT THE TUTORIAL
This tutorial shows how to print values of variables on the evive TFT screen (just like serial print in arduino), making projects more portable as one does not have to connect the USB to debug his/her code.
TUTORIAL INFO
Programming Platform: Arduino IDE Type: Tutorial Difficulty: Medium Estimated Time: 2 hours

Introduction

The function to print values on TFT Display is used in same way as the serial.print/println() commands in Arduino.

You can download the library for the same here

Following are the steps for implementation:

  • Initialise object Tftserial (before setup) :

    • Tftserial ser = Tftserial();

  • Begin serial (in setup)

    • ser.begin();

  • Use (in loop): The syntax is similar to Arduino serial print :

    • ser.print() or ser.println() the input can be char/int or string only.

WARNING : if Tftserial is initialised or even if object is declared then then tft should not be used for any other purpose. (i.e even tft object should not be initialised by the user) as it might cause unpredictable behaviour of the screen due to clash betwwen the tft object by user and the one initialised internally in the library
